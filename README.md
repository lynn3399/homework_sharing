# 体系结构作业——Python实现记分牌调动

## 可优化的点
1. 输入：由于这是为完成作业而完成的作业，所以对输入的判定并没有做，一旦输入错误程序会自动结束。当前能够接受的输入为全大写输入，示例如下
LD F6 34+ R2
LD F2 45+ R3
MULT F0 F2 F4
SUBD F8 F6 F2
DIVD F10 F0 F6
ADDD F6 F8 F2

2. 可以增加一个自动执行到结束的按钮，设置周期之间的间歇时间
